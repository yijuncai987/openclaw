# WeChat Xiangcai Style

## Overview

为“香菜博士”公众号撰写文章时，采用轻松、直接、带个人视角的表达，偏实用/观点型内容，结构清晰，段落短。

## Style Guidelines

- **人称与语气**：第一人称为主（“我觉得/我认为/我这周才开始…”），语气轻松直给，不端着。
- **结构节奏**：短段落 + 明确层级；常用“先说背景 → 重点来了 → 列举/清单 → 总结”。
- **观点表达**：敢下判断，但会给理由；偏“经验总结/观察预判”。
- **连接词**：多用“所以/但/不巧的是/实际上/总结下”。
- **标题风格**：带数字或强引导（如“X个原因/一个小细节/推荐一种最爽的方法/末尾有惊喜”）。
- **列表方式**：编号列点（第一、第二…）或短小条列，强调“关键点”。
- **口语化提示**：加入“重点来了/继续看/看下图/这个事”的节奏词。
- **轻度情绪与比喻**：可用“像/更像/本质是”来做类比；偶尔带一点夸张但不浮夸。
- **结尾**：总结一段 + 行动号召（评论、关注、留言）。

## Writing Workflow

1. **开头 2-4 句**：抛出背景/现象 + 个人动机/观察。
2. **中段观点**：用“我终于知道/重点来了/本质是…”引出核心观点。
3. **清单段**：2-8 条的编号/列表，段落短，句子直。
4. **总结段**：用“总结下/所以/本质是…”收束。
5. **行动号召**：引导互动或关注。
