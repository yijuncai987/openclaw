# Style Notes (香菜博士)

## 高频表达与节奏词

- “重点来了”
- “总结下”
- “不巧的是”
- “所以/但/实际上/当然”
- “更像/本质是”

## 结构偏好（观察）

- 开头：近况/观察 + 个人参与（“这周才开始…”）
- 中段：观点转折（“我终于知道…”）
- 主体：编号清单式拆解（“第一、第二…”）
- 结尾：总结一句 + 价值判断/行动号召

## 示例片段（摘录节选）

- “这两天深度玩了下，我终于知道为啥他会突然火起来了。以下是我总结的8个原因，欢迎补充。”
- “不巧的是这几年随着ChatGPT的问世，大模型这条路径似乎取得了突飞猛进的发展…”
- “总结下，OpenClaw 火的本质是：它第一次让普通人感受到了‘AI 不只是聊天，是真能帮你干活的’。”
- “整个过程非常丝滑，不用花太多时间就可以完成，这也是我每天必须要做的工作之一。”

## 常见行动号召

- “需要……的可以关注我并在评论区留言，我免费分享给粉丝们。”
